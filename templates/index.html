Welcome to my LoTR App!

Instructions: Hello! And welcome to my Lord of the Rings web application! Here, you will find three buttons which can be interacted with by clicking them.

The first two buttons will display random character and quote information to the screen, respectively. If you click the button again, the currently displayed info will be cleared and the new info will be displayed instead.

The third button will clear the screen of all text.

The character info is from the books, while the quotes are from the LoTR movies (including the Hobbit series).

This information is available via a public API: https://the-one-api.dev/ Note: this public API limits API calls to 100 every 10 minutes, so be careful when clicking very fast! Have fun and enjoy!
